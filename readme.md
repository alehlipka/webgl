## Installation

```bash
npm install
```

## Commands

```bash
npm run dev     # run app in dev mode
npm run build   # build app
npm run preview # run builded app
npm run test    # run cli tests
npm run test-ui # run tests gui
```

## Linter

```bash
npx eslint .
npx eslint . --fix
```

## Prettier

```bash
npx prettier . --check
npx prettier . --write
```
